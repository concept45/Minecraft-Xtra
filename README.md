<h1>Minecraft Xtra ?</h1>
<p>Minecraft Xtra un site web permettant de gerer l'économie de votre serveur facilement.</p>

<h1>Fonctionnalité ?</h1>
<p>Minecraft Xtra comporte les fonctionnalités suivante :</p>
<p>-Envoie d'argent entre utilisateur</p>
<p>-Dons par Allopass</p>
<p>-Achat de permissions</p>
<p>-Gestion des packs de texture</p>
